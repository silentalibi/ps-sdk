PhotoshopSDK version 1.1
API and Architecture: Daniel Koestler (http://blogs.adobe.com/koestler)
Encryption and Architecture: Renaun Erickson (renaun.com)

Please see the ASDocs in the documentation folder for more information.

Visit the Adobe Photoshop Developer Center to learn how to use the SDK in your own AIR applications: https://www.adobe.com/devnet/photoshop.html

Required libraries:
	* as3corelib (https://github.com/mikechambers/as3corelib)
	* as3crypto (https://code.google.com/p/as3crypto/)
	* StructuredLogTestingSDK (https://code.google.com/p/structuredlogtesting/)
	
If you're compiling the PhotoshopSDK yourself, you can either include the source paths from the above libraries, or drop the SWC files into the "libs" folder of your project.

Code is in src/com/adobe/photoshop/
The SWC is bin/PhotoshopSDK.swc
