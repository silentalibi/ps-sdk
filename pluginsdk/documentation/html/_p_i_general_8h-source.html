00473 //  These structures obey Macintosh 68k alignment and padding rules though
00474 //  generally they are laid out so fields have natural alignment and any
00475 //  needed padding is explicit in the structure.
00476 //
00477 //  In addition to these structure types, simple types such as int32, int16,
00478 //  etc. are used. Two other examples are PString and FlagSet.

00481 //  be present for all 255 characters.  The initial length byte is present
00482 //  in the property.

00485 //  represented by the most significant bit of the first byte, the eighth
00486 //  member is in the least significant bit of the first byte, etc.
00487 //  The length of the set is in the property's length field.
00488 //
00489 //  Arrays are also used in places with bounds either taken from the
00490 //  property length or explicitly represented within the property.
