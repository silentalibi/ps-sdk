Details of tools used to build Photoshop and the SDK examples

It is recommended that third party developers use the same tools that we use to build Photoshop.

Below is the information regarding the Windows and Macintosh platforms. This is also noted in the What's New section of the Photoshop SDK HTML documentation

Photoshop 25.1 - 2024

Windows was built on a Windows 10 64 bit machine using the 64 bit tool chain and the following version: Microsoft Visual Studio 2022 v143 Version 17.0.5 Windows 10 SDK version 10.0.22621.0

OS X was built on an OS X 12.6.5 machine Using Xcode 14.2 (14C18), Base SDK 12.1, OS X Deployment Target of 11.0

Photoshop 24.1 - 2023

Windows was built on a Windows 10 64 bit machine using the 64 bit tool chain and the following version: Microsoft Visual Studio 2022 v143 Version 17.0.5 Windows 10 SDK version 10.0.20348.0

OS X was built on an OS X 11.6.5 machine Using Xcode 13.2.1 (13C110), Base SDK 12.1, OS X Deployment Target of 11.0

Photoshop 23.4 - 2022

Windows was built on a Windows 10 64 bit machine using the 64 bit tool chain and the following version: Microsoft Visual Studio 2019 v142 Version 16.9.16 Windows 10 SDK version 10.0.19041.0

OS X was built on an OS X 10.15.7 machine Using Xcode 12.4 (12D4e), Base SDK 11.0, OS X Deployment Target of 10.15

Photoshop 22.3 - 2021

Windows was built on a Windows 10 64 bit machine using the 64 bit tool chain and the following version: Microsoft Visual Studio 2019 v142 Version 16.5.4 Windows 10 SDK version 10.0.18362.0

OS X was built on an OS X 10.15.7 machine Using Xcode 12.2 (12B45b), Base SDK 11.0, OS X Deployment Target of 10.14

Photoshop 22.0 - 2021

Windows was built on a Windows 10 64 bit machine using the 64 bit tool chain and the following version: Microsoft Visual Studio 2019 v142 Version 16.5.4 Windows 10 SDK version 10.0.18362.0

OS X was built on an OS X 10.14.4 machine Using Xcode 11.3 (11C29), Base SDK 10.15, OS X Deployment Target of 10.14

Photoshop 21.0 - 2020
