Adobe Photoshop SDK File List

Here is a list of all files with brief descriptions:
JSScriptingSuite.h [code]This file contains all the public definitions and structures related to using the JavaScript engine
PIAbout.h [code]This file describes Photoshop's plug-in module interface for the about box selector
PIAcquire.h [code]This file describes Photoshop's Acquisition/Import module interface
PIActions.h [code]This file contains the public definitions and structures used by all plug-ins for scripting control and access
PIActionsPlugin.h [code]This file describes Photoshop's actions plug-in module interface
PIAliasSuite.h [code]This file describes the Photoshop Alias Suite for plug-ins
PIBufferSuite.h [code]This file describes Photoshop's Buffer Suite for plug-ins
PIChannelPortOperations.h [code]This file describes operations Photoshop can perform via the Channel Ports Suite
PIChannelPortsSuite.h [code]This file describes Photoshop's Channel Ports Suite for plug-ins
PIColorSpaceSuite.h [code]This file describes Photoshop's ColorSpace Suite for plug-ins
PIErrorSuite.h [code]This file contains the public definitions and structures used by plug-ins to manage asserts, debugging, and returning errors to the host
PIExport.h [code]This file describes Photoshop's Export plug-in module interface
PIFeatureSuite.h [code]This file describes Photoshop's Feature Suite for plug-ins
PIFilter.h [code]This file describes Photoshop's Filter module interface
PIFormat.h [code]This file describes Photoshop's Image Format plug-in module interface
PIGeneral.h [code]This file contains the public definitions and structures used by all plug-ins
PIGetFileListSuite.h [code]This file contains all the public definitions and structures for GetFileList Suite
PIGetPathSuite.h [code]This file contains all the public definitions and structures related to user interface elements
PIHandleSuite.h [code]This file describes Photoshop's Handle Suite for plug-ins
PIPatternsSuite.h [code]This file describes Photoshop's Patterns Suite for plug-ins
PIPicker.h [code]This file describes Photoshop's Color Picker plug-in module interface
PIProgressSuite.h [code]This file describes Photoshop's Progress Suite for plug-ins
PIProperties.h [code]This file contains the public definitions and structures for the properties callback suite
PIResDefines.h [code]This file contains the public definitions used to create PiPL resources with the PiPL master template, PIGeneral.r and PIGeneral.h
PISelection.h [code]This file describes Photoshop's Selection plug-in module interface interface
PIStringTerminology.h [code]This file defines constants for the strings used to access descriptor events, keys, classes, enum types, and enum values
PITerminology.h [code]This file defines constants for the keys used to access descriptor events, keys, classes, enum types, and enum values
PITypes.h [code]This file describes the type definitions used in Photoshop's interface files
PIUIHooksSuite.h [code]This file contains all the public definitions and structures related to user interface elements
PIUXPSuite.h [code]This file contains the public definitions and structures used by plug-ins wishing to communicate with UXP based Plugins
